# HIEUNGOCTRA - Website Bán Trà

Dự án website bán trà full-stack gồm frontend và backend tách riêng.

## Công nghệ sử dụng

- Frontend: React, Vite
- Backend: Java Spring Boot
- Database: MySQL
- Bảo mật: Spring Security

## Chức năng chính

### Người dùng

- Xem danh sách sản phẩm
- Xem chi tiết sản phẩm
- Tìm kiếm và lọc sản phẩm
- Quản lý giỏ hàng
- Đặt hàng
- Quản lý tài khoản cá nhân
- Xem đơn hàng cá nhân

### Quản trị viên

- Quản lý sản phẩm
- Quản lý người dùng
- Quản lý đơn hàng

## Cấu trúc thư mục

HIEUNGOCTRA/

- back-end/
- font-end/

## Cách chạy frontend

```bash
cd font-end
npm install
npm run dev
```

## Cách chạy backend

```bash
cd back-end
mvnw spring-boot:run
```
